# High School Python Archive

Hey! This is just a quick repository to save all the old Python files I worked on back in high school. I wanted to store them in one clean place before I start uploading my actual college computer science projects. It is pretty cool looking back to see how my code slowly got better over time.

## Directory Breakdown

### 1. Tutoring Stuff (tutoring-lessons/)
These are my very first practice scripts from day 1 to day 10. My tutor had me running really basic exercises here just to help me wrap my head around fundamental programming concepts. It is mostly simple things like testing variable syntax, basic loops, print statements, and standard if/else conditions.

### 2. Homework Problems (assignments/)
I used these files as independent worksheets to test myself on what I learned during my tutoring lessons. A highlight in here is assignment 5, which is a text-based RPG game that runs over 200 lines long. I spent a lot of time mapping out all the different story paths and choice loops for that one.

### 3. Clone Projects (games-and-projects/)
I also wanted to experiment with visual layouts and player inputs, so I tried putting together a few classic games. To be completely upfront, I did not build the core code for pacman.py, tetris.py, or the snake game (test.py) entirely from scratch. I followed online guides and adapted their templates so I could analyze how game loops handle graphics and keyboard controls.
